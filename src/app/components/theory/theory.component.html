Angular Complete Tutorial

1.Angular Basics

1.What is Angular?

Angular is a JavaScript based client-side development framework which is used to build the Single Page Applications (SPA) for Mobile and Desktop.

2.What is a Framework?

A framework is a platform for developing Software Applications.

A framework can have pre-defined classes and functions that can be re-used to add several functionalities, which otherwise we would have to write it from scrach by our own.

3.What is a Single Page Application (SPA)?

A Single Page Application (SPA) is a web application, which has only one HTML page. When we navigate around, only the content of the HTML page changes. The page itself never changes.

4.What is the command to check the node version?

Command : node -v

5.What is the command to check the npm version?

Command : npm -v

6.What is the command to install Angular CLI?

Command : npm i -g @angular/cli@latest

6.What is the command to check the Angular version?

Command : ng v

7.What is the command to create a new angular project?

Command : ng new <Project_Name>

8.What is the command to run the angular application?

Command : ng serve

9.What is the default port number to run the Angular Application?

port : 4200
